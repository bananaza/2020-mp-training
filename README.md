# 2020专业实习——微信小程序
## 一.项目说明

这是一个基于mpvue框架+SpringBoot服务端实现的小型校园跳蚤市场小程序，由于开发周期较短，API文档可能有不完善的地方。

## 二.文件说明

mp-project为小程序项目文件，shopping为SpringBoot项目文件，开发文档内包含API文档、数据库设计文档及一些功能文档总结（里面的static文件夹为开发时保存上传图片文件所用）。

## 三.关于

开发于2020年6月10日，后续计划将此项目移植至uniapp。
